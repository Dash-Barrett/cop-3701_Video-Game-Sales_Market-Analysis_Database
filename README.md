# cop-3701_Video-Game-Sales_Market-Analysis_Database
Project for COP 3701

The main priority of this project is to design a database in which information regarding video game sales can be stored and accessed. Ideally, information such as names and region sales would be able to be stored and accessible whenever desired, while also being able to add and adjust this information whenever neccessary.

The end users of this database would likely be game data analysts and marketing analysts.

Oracle Database will be the relational database management system (RDBMS) used in this project, while DataGrip and Visual Studio Code will be the softwares that all of the scripts are made on.

Data Source: https://www.kaggle.com/datasets/gregorut/videogamesales
